# option

### 平价公式

\[
c + Ke^{-rT} = p + S_0
\]

其中：

- (c)：欧式看涨期权当前价格
- (p)：欧式看跌期权当前价格
- (S_0)：标的资产当前价格
- (K)：执行价
- (T)：到期时间
- (r)：无风险利率
- (Ke^{-rT})：现在买一张到期时恰好变成 (K) 的无风险债券

------



**如果两个组合在未来每一种状态下的到期收益完全相同，那么今天的价格必须相同。**

**若今天不相等就会出现套利**



------

# 情况 1：(S_T > K)

## 组合 A：(c + Ke^{-rT})

- 看涨期权价值：(\max(S_T-K,0)=S_T-K)
- 债券到期给你：(K)

所以组合 A 到期总价值：

[
(S_T-K)+K=S_T
]

## 组合 B：(p + S_0)

这里看的是到期价值，不是现在价格：

- 股票到期价值：(S_T)
- 看跌期权价值：(\max(K-S_T,0)=0)

所以组合 B 到期总价值：

[
S_T+0=S_T
]

------

# 情况 2：(S_T < K)

## 组合 A

- 看涨期权价值：0
- 债券到期给你：(K)

所以组合 A 到期总价值：

[
0+K=K
]

## 组合 B

- 股票到期价值：(S_T)
- 看跌期权价值：(K-S_T)

所以组合 B 到期总价值：

[
S_T+(K-S_T)=K
]

------

# 情况 3：(S_T = K)

两边显然都是 (K)。

------

## 三、结论：两组合到期收益逐状态完全相同

所以无论到期时股价涨、跌还是正好等于执行价：

[
\text{组合A到期价值}=\text{组合B到期价值}
]

更紧凑地写：

[
\max(S_T-K,0)+K=\max(K-S_T,0)+S_T
]

因此根据**无套利原则**，今天它们价格必须相同：

[
c + Ke^{-rT} = p + S_0
]

这就是平价公式。

------

# 四、为什么“不相等”就会出现套利

这一步才是“从可以套利的假设上”去理解。

------

## 假设 1：如果

[
c + Ke^{-rT} > p + S_0
]

说明 **组合 A 比组合 B 贵**。

但两者到期收益一模一样，所以贵的那个被高估了，便宜的那个被低估了。

这时可以：

- **卖空组合 A**
- **买入组合 B**

也就是：

- 卖出看涨期权
- 借出/卖出债券那一边对应头寸
- 买入看跌期权
- 买入股票

你在今天会先收到一笔正现金流：

[
(c + Ke^{-rT}) - (p + S_0) > 0
]

而到期时，因为两组合收益完全相同，一个多头一个空头正好抵消：

[
\text{到期净收益}=0
]

于是你实现了：

- 今天净赚一笔钱
- 未来无风险、无成本

这就是套利，不允许存在。

所以不可能有：

[
c + Ke^{-rT} > p + S_0
]

------

## 假设 2：如果

[
c + Ke^{-rT} < p + S_0
]

那就反过来：

- 买入组合 A
- 卖空组合 B

今天仍然能收到：

[
(p + S_0) - (c + Ke^{-rT}) > 0
]

到期净收益仍然为 0。

这也是套利，所以也不可能存在。

------

## 因此只能有

[
c + Ke^{-rT} = p + S_0
]

------
