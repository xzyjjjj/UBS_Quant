1️⃣ 收益 / 概率类

# 一、收益 / 概率类参数

这类指标主要用来判断 **策略胜率和收益空间**。

### 标记价 (Mark Price)

交易所估算的 **理论成交价**。

通常公式类似：

[
mark = \frac{bid + ask}{2}
]

作用：

* 防止操纵
* 用于保证金计算
* 用于风险计算

不是一定能成交的价格。

---

### 卖方回报率

**卖期权的收益率**

例如：

卖一个期权

* 收到权利金 = 2
* 占用保证金 = 10

收益率

[
Return = 2/10 = 20%
]

一般用于：

* 卖Call
* 卖Put
* Wheel策略

---

### 盈利概率 (Probability of Profit)

理论模型算出来的 **策略盈利概率**。

例如：

* 盈利概率 65%

意思是：

模型认为
**65%概率不会亏钱**

通常是用：

* Black-Scholes
* Monte Carlo

算的。

---

### 盈亏平衡点 (Break Even)

期权策略 **开始赚钱的价格**。

例如：

Call

[
BreakEven = Strike + Premium
]

例子：

* strike = 100
* premium = 3

[
BE = 103
]

---

# 二、Greeks（风险指标）

这些是 **期权最核心的东西**。

---

## Delta

\[
\Delta = \frac{\partial Option}{\partial S}
\]

意思：

**标的涨1块钱，期权涨多少**

例子

| Delta | 含义            |
| ----- | ------------- |
| 0.5   | 股票涨1 → 期权涨0.5 |
| 1     | 类似股票          |
| 0     | 几乎不动          |

经验：

| Delta | 意义   |
| ----- | ---- |
| 0.1   | 深度虚值 |
| 0.5   | 平值   |
| 0.9   | 深度实值 |

---

## Gamma

\[
\Gamma = \frac{\partial \Delta}{\partial S}
\]

意思：

**标的涨1块，Delta变化多少**

Gamma高说明：

* Delta变化快
* 风险大

通常：

**平值期权 Gamma最大**

---

## Theta

[
\Theta = \frac{\partial Option}{\partial t}
]

时间损耗。

通常：

**每天损失多少价值**

例如：

Theta = -0.05

意思：

每天亏

[
0.05
]

卖期权的人最喜欢 Theta。

---

## Vega

[
Vega = \frac{\partial Option}{\partial \sigma}
]

波动率变化对期权的影响。

例如：

Vega = 0.1

IV上涨1%

期权涨

[
0.1
]

---

## Rho

利率敏感度。

利率涨1%

期权涨多少。

实际交易中：

**几乎没人看Rho**。

---

# 三、波动率

### 隐含波动率 (Implied Volatility, IV)

这是 **期权最重要指标之一**。

含义：

市场认为未来价格波动多大。

例如：

IV = 80%

意味着：

市场认为一年内波动

±80%

---

你图里有几个特别夸张的：

```
IV 2301%
IV 1162%
```

说明：

**流动性极差 / 报价异常**

基本没人交易。

---

# 四、市场报价

这些是 **真实盘口数据**。

---

### 最新价 (Last)

最近一笔成交价。

---

### 买入价 (Bid)

有人愿意买的价格。

---

### 卖出价 (Ask)

有人愿意卖的价格。

---

### 中间价 (Mid)

[
Mid = (Bid + Ask)/2
]

理论成交价。

---

# 五、交易活跃度

### 成交量 (Volume)

今天成交多少张。

例子：

```
volume = 241
```

说明：

今天成交241张期权。

---

### 未平仓数 (Open Interest)

市场 **还没平仓的合约数量**。

例如：

```
OI = 5000
```

说明：

市场还有5000张期权没平掉。

---

区别：

| 指标     | 含义   |
| ------ | ---- |
| Volume | 今天成交 |
| OI     | 市场存量 |

---

# 六、合约信息

### 行权价 (Strike)

期权执行价格。

例如：

```
strike = 10
```

Call：

可以买10。

Put：

可以卖10。

---

### Call / Put

左边：

Call

右边：

Put
