\section{Week 2：数据清洗与特征工程}

本周目标：在统一面板（\texttt{output/market\_panel/processed/panel.csv}）上完成预处理，并构建可复用的特征列（特征实现与数据拉取解耦，避免每次加特征都重新下载）。

\subsection{数据处理}
\begin{itemize}
  \item \textbf{缺失值插值}：面板外连接后，对连续缺失做前向填充（\texttt{ffill}），保证宏观/利率类序列在交易日可用。
  \item \textbf{异常值 IQR}：对特征列可采用 IQR 截尾（winsorize），将极端值裁剪到 $[Q1-1.5\mathrm{IQR},\, Q3+1.5\mathrm{IQR}]$，降低尖峰对模型/回测的影响。
\end{itemize}
