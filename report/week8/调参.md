## 调参

**涉及的参数：**

| 参数                | 含义                                  | 作用方向                                   |
| ------------------- | ------------------------------------- | ------------------------------------------ |
| `hidden`            | MLP 隐藏层宽度                        | 大一些容量更强，但更易过拟合               |
| `lr`                | 学习率                                | 大了收敛快但不稳，小了稳但慢               |
| `batch-size`        | 每步训练样本数                        | 大 batch 更平滑，小 batch 噪声更大但可泛化 |
| `epochs`            | 训练轮数                              | 多轮可降训练误差，但可能过拟合             |
| `residual-cap`      | 残差限幅（`cap*tanh(raw/cap)`）       | 小 cap 更稳、更保守；大 cap 拟合更强       |
| `huber-delta`       | Huber 损失拐点                        | 小值更抗异常点；大值更接近 MSE             |
| `vega-weight-floor` | Vega 权重下限                         | 防止小 Vega 样本权重爆炸                   |
| `vega-weight-power` | Vega 加权幂次（`w~1/vega^power`）     | 越大越强调低 Vega 样本                     |
| `mono-penalty`      | 对 `K` 单调性违反的惩罚强度           | 大则更少单调套利，但拟合可能变差           |
| `convex-penalty`    | 对 `K` 凸性违反的惩罚强度             | 大则曲面更平滑合理，但可能欠拟合           |
| `calendar-penalty`  | 对期限单调性违反的惩罚强度            | 大则更少日历套利，但会压制灵活性           |
| `strike-bin-size`   | 按行权价分组粒度（做 calendar 检查）  | 小更精细严格，大更粗糙稳健                 |
| `ttm-bin-size`      | 按期限分组粒度（做 mono/convex 检查） | 小更精细严格，大更宽松稳健                 |
| `train-ratio`       | 训练集时间占比                        | 大则训练样本更多                           |
| `val-ratio`         | 验证集时间占比                        | 大则调参更稳，但训练样本更少               |
| `max-train`         | 训练样本抽样上限                      | 控制训练时间与稳定性                       |
| `max-val`           | 验证样本抽样上限                      | 控制评估时间与方差                         |
| `max-test`          | 测试样本抽样上限                      | 控制评估时间与方差                         |
| `seed`              | 随机种子                              | 影响采样与初始化，可做稳健性检查           |





**参数到更优参数（run_0100）的关键变化：**

| 参数                 | 默认           | 更优方向（本次） | 作用                                 |
| :------------------- | :------------- | :--------------- | :----------------------------------- |
| hidden               | 24             | 48               | 提升模型容量，残差拟合更充分         |
| lr                   | 0.0025         | 0.0035           | 收敛更快更到位（在 batch=64 下稳定） |
| batch-size           | 64             | 64               | 这个保持不变，表现稳定               |
| epochs               | 8              | 8                | 不需要拉太长，避免过拟合             |
| residual-cap         | 3              | 12               | 放宽残差限幅，避免欠拟合             |
| huber-delta          | 0.5            | 0.3              | 更抗极端点噪声                       |
| vega-weight-power    | 1.0            | 0.5              | 减弱过强加权，提升泛化               |
| mono/convex/calendar | 0.05/0.02/0.02 | 0/0/0            | 本批数据上约束会拉高 test 误差       |
| strike-bin-size      | 2.5            | 5.0              | 更粗分组，减少约束噪声影响           |
| ttm-bin-size         | 0.02           | 0.05             | 同上，更稳                           |
